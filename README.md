# toeplitz
A program that finds inscribed squares in Jordan curves.

To use: 

1. Run the Java code.
2. Press and hold the mouse to draw a curve.
3. Release the mouse to finish the curve.
4. Wait as the computer finds squares inscribed in the curve.
